
# How It's Made:
### Tech used: 
HTML, CSS, JS, EJS, MongoDB, PassPort & Mongoose
![anime](anime1.png)




### Lessons Learned:
Learned how to incorporate iframe and regex to allow the video url to be hosted appropriately.

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `npm run savage`
2. Navigate to `localhost:8080`
